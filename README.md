# 🕯️ Aroma Home — Artisan Soy Candles (Dublin)

A full-featured e-commerce web application built with **Flask**, designed for a handmade candle shop based in Dublin.

## 📊 Project Statistics
* **Total Lines of Code:** ~1,000+ lines of Python, HTML, and CSS.
* **Backend Logic:** 400+ lines in `app.py`.
* **Complexity:** 10+ custom templates, Stripe integration, and full database CRUD.

## ✨ Key Features
* **🛒 Shopping Cart:** Users can add, update, and remove candles seamlessly.
* **💳 Stripe Integration:** Secure payment processing in test mode.
* **🔐 Auth System:** Full user registration and login system.
* **📊 Admin Panel:** Dedicated dashboard to manage inventory and view orders.

## 🛠 Tech Stack
* **Backend:** Python / Flask
* **Database:** SQLite / SQLAlchemy
* **Frontend:** HTML5, CSS3, Bootstrap 5
* **Payments:** Stripe API

## 🚀 How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run the app: `python app.py`
